# PSA App Theme Spec - Clean Retro Future

## Core Direction
A clean iPhone-friendly app inspired by late 90s and early 2000s anime web pages, desktop UI chrome, and sci-fi operating system visuals.

This should feel like:
- Netscape / Internet Explorer / old Windows shell influence
- anime-tech and cyber UI mood
- nostalgic without becoming ugly or hard to use
- premium collector tool, not novelty toy

## Design Rule
Use nostalgic styling for the shell.
Keep the interaction model modern.

Target split:
- 80% modern usability
- 20% retro skin

## Visual Pillars

### 1. Browser chrome framing
- app hero should feel like a compact browser window
- metallic/silver top bar
- cobalt blue title strip accents
- subtle inset borders

### 2. Soft sci-fi backdrop
- pale sky blue, icy white, chrome gray, deep cobalt
- faint glow or brushed-light gradients
- optional subtle grid / scanline / cloud texture at very low opacity

### 3. Desktop-era buttons and panels
- bevel effect
- slightly raised controls
- inset fields
- strong border hierarchy
- crisp corner radii, not overly round

### 4. Anime web energy without clutter
- no giant character art inside primary workflow
- references should come through color, chrome, labels, tiny badges, and section framing
- optional decorative background watermark later

## Color Palette
- Deep Cobalt: `#1846d8`
- Electric Blue: `#3f7cff`
- Soft Sky: `#d9ecff`
- Ice White: `#f7fbff`
- Chrome Silver: `#cfd6e4`
- Steel Border: `#7f90b2`
- Midnight Navy: `#0d1630`
- Success Green: `#30c96b`
- Warning Gold: `#e6bf3f`
- Alert Red: `#d95d73`

## Typography
- Body: system UI font stack for readability on iPhone
- Accent labels: optional retro feeling through uppercase spacing and compact sizing
- avoid full bitmap font for core inputs and numbers

## Component Styling

### Hero/Header
- browser-like chrome top strip
- page title inside windowed shell
- optional fake window lights/buttons kept subtle

### Panels
- frosted white or pale silver cards
- inset inner line
- darker title strip or micro label at top

### Inputs
- white/ice backgrounds
- dark text
- inset bevel border
- clear touch targets

### Buttons
- silver or blue desktop-inspired gradient
- pressed effect on active state
- strongest CTA can be cobalt

### Stat Cards
- more dashboard-like
- metallic edge + white face
- numbers remain bold and clean
- keep recommendation pills vivid

### History/Watchlist
- should feel like saved records in an old desktop utility
- compact but touch-friendly

## Texture Rules
- subtle only
- avoid noisy backgrounds behind important text
- use texture in shell, not in form fields

## iPhone UX Constraints
- preserve strong spacing and thumb-friendly controls
- no tiny pixel text for real actions
- no fake window chrome so large it wastes vertical space
- maintain fast scanability

## Immediate Reskin Plan
1. convert dark mode shell to bright retro-future shell
2. add browser-window hero styling
3. update cards and buttons to bevel/chrome treatment
4. preserve app structure and all functionality

## Later Optional Enhancements
- tiny retro icon set for sections
- theme toggle: modern / retro-future
- subtle wallpaper mode behind app shell
- collectible-card scanline inspection overlay
